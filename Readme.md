# Лабораторная работа №3

## Тема
Работа с массивами, указателями, динамической памятью и строками

## Цель работы
Закрепить на практике:
- работу со статическими и динамическими массивами;
- использование указателей;
- управление динамической памятью;
- работу со строками в стиле C (char*);
- создание модульной программы с меню и обработкой ошибок ввода.

---
### Сборка 

``` bash
cmake -S . -B build
cmake --build build
```

### Запуск
``` bash
./build/Lab3
```
